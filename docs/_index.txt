Robot Control - Quick Reference
============================================================

MOST COMMON COMMANDS:

  move forward <meters>                    Move robot forward
  turn clockwise <degrees>                 Turn robot clockwise
  robot stop                               Stop robot immediately
  config list                              Show all settings
  map save <filename>                      Save current map

LAUNCH & MAPPING:

  launch start <type> ...                  Start ROS2 launch (map, teleop, etc)
  launch list                              Show available launch types
  launch kill <type>                       Stop a launch process
  map list                                 List saved maps

SCRIPT COMMANDS:

  script square <meters>                   Drive in a square pattern
  script rotate_stress                     Run rotation stress test
  script circle_stress <diameter>          Run circle stress test

ABBREVIATIONS:
  Commands: m (move), t (turn), r (robot), c (config), lch (launch), sys (system), scr (script)
  Subcommands: fwd (forward), bak (backward), clk (clockwise), ccw (counterclockwise), lst (list), etc.
  Examples: m fwd 1.5  |  t clk 90  |  c lst  |  r stp

For detailed help on any command:
  help <command>

Examples:
  help move forward
  help config set
